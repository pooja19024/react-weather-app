# 🌦️ React Weather App

A simple and responsive weather application built using React that fetches real-time weather data using an API.

---

## 🚀 Features
- Search weather by city name
- Shows temperature, weather condition, and feels like
- Error handling for invalid city
- Clean UI using Material UI

---

## 🛠️ Tech Stack
- React.js
- JavaScript
- Weather API
- Material UI
